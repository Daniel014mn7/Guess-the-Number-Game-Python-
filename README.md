#  Guess-the-Number Game

A simple Python game where the user tries to guess a randomly generated number.  
This project demonstrates basic Python concepts such as loops, conditionals, user input, and random number generation.

---

# Features
- Generates a random number between a chosen range
- Tells you if your guess is too high or too low
- Counts the number of attempts
- Simple, clean, and beginner-friendly code

---

# Skills Demonstrated
- Python basics
- Loops ('while')
- Conditional statements ('if/elif/else')
- Input handling
- Using the 'random' library
